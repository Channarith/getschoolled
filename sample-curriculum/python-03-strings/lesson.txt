LESSON: Python - Chapter 3: Working with Strings
LANGUAGE: en

SLIDE 1 | Creating Strings
A string is a sequence of characters in single or double quotes. Triple quotes
let a string span several lines, which is handy for longer text.

  greeting = "Hello"
  paragraph = """line one
  line two"""

NARRATION: Strings use single, double, or triple quotes; triple quotes span lines.

SLIDE 2 | Strings are Immutable
Once created, a string cannot be changed in place. Operations that look like they
edit a string actually build and return a new string instead.
NARRATION: Strings never change in place; string operations return brand-new strings.

SLIDE 3 | Indexing
Each character has a position, starting at zero for the first character. Negative
indexes count from the end, so index minus one is the last character.

  word = "Python"
  word[0]    # 'P'
  word[-1]   # 'n'

NARRATION: Indexing starts at zero, and negative numbers count from the end.

SLIDE 4 | Slicing
A slice extracts part of a string using start and stop positions. The start is
included and the stop is excluded.

  word[0:3]  # 'Pyt'

NARRATION: A slice grabs a range; the start is included and the stop is excluded.

SLIDE 5 | Concatenation and Repetition
The plus sign joins strings together, and the star repeats a string a given
number of times.

  "ab" + "cd"   # 'abcd'
  "ha" * 3      # 'hahaha'

NARRATION: Plus joins strings; star repeats a string several times.

SLIDE 6 | Common String Methods
Strings come with helpful methods such as upper, lower, strip, and replace. Each
returns a new string and leaves the original unchanged.

  "  Ada ".strip()        # 'Ada'
  "cat".replace("c","b")  # 'bat'

NARRATION: Methods like upper, lower, strip, and replace return new strings.

SLIDE 7 | Splitting and Joining
The split method breaks a string into a list of pieces, and join combines a list
of strings back into one string with a separator.

  "a,b,c".split(",")     # ['a','b','c']
  "-".join(["a","b"])    # 'a-b'

NARRATION: split breaks text into a list; join glues a list back into text.

SLIDE 8 | f-strings
An f-string lets you insert values directly into text by putting an f before the
quotes and variables inside curly braces. This is the clearest way to build
messages.

  name = "Ada"
  print(f"Hello, {name}!")

NARRATION: f-strings drop variables right into text using curly braces.

SLIDE 9 | Length and Membership
The len function returns how many characters a string has, and the in keyword
checks whether one string appears inside another.

  len("Python")     # 6
  "th" in "Python"  # True

NARRATION: len counts characters, and the in keyword tests if text is contained.

FACT: Strings in Python are immutable; methods return new strings.
FACT: String indexing starts at zero, and negative indexes count from the end.
FACT: A slice includes the start position and excludes the stop position.
FACT: f-strings insert variables into text using curly braces.
FACT: The split method turns a string into a list, and join turns a list into a string.
FACT: The len function returns the number of characters in a string.
