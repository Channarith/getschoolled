AI CLASSROOM - BRANDING
=======================
Plain text on purpose (repo no-markdown house rule).

-----------------------------------------------------------------------------
1. LOGO
-----------------------------------------------------------------------------
Concept: a minimalist, single-color mark in the spirit of Apple / X / Micron.
A graduation mortarboard whose tassel curls back into the bowl of a capital G
- the cap is the G's top stroke, the tassel curl is the body. Reads as
"graduation" at favicon size and as a clean monogram at any scale. No
gradients, no detail, one solid silhouette.

Mascot/design guardrail:
  - Keep mascot and theme art secular and classroom-focused.
  - Avoid ceremonial monuments, faith-based symbols, or devotional visual cues.
  - For Khmer-language or Cambodian audience designs, use neutral language,
    family, study, food, music, pattern, landscape, or everyday-learning motifs.

Sources (vector, single-source-of-truth):
  docs/brand/aiclassroom_mark.svg          icon (square, viewBox 0 0 256 256)
  docs/brand/aiclassroom_wordmark.svg      horizontal lockup (mark + text)


Both SVGs use fill="currentColor" so a single file inverts cleanly on light
or dark backgrounds via CSS (`color: ...` on the parent / inline style).

Generated raster assets (do NOT edit by hand; regenerate via
`python3 scripts/build_brand_assets.py`):
  docs/brand/aiclassroom_logo.png                1024x1024 color on navy
  docs/brand/aiclassroom_logo.webp               1024x1024 color on navy
  docs/brand/aiclassroom_logo_binary.png         1-bit Floyd-Steinberg dither,
                                                  for grayscale/halftone print
  docs/brand/aiclassroom_logo_binary_threshold.png  1-bit hard threshold,
                                                  for stamps / screen print /
                                                  single-ink / laser engraving

Web assets (apps/web/public/, served from /):
  /favicon.ico       multi-size browser icon (16/32/48/64)
  /logo-mark.webp    128x128 nav icon
  /logo-mark.svg     same as the source SVG (themable via currentColor)
  /logo.webp         512x512 hero / share / OG-image
  /wordmark.webp     1024x224 horizontal lockup (header / press kit)

Brand colors (UI uses CSS vars; mark uses solids):
  navy    #0b1020   primary background
  light   #e8ecf6   primary foreground / mark color on dark
  accent  #0ea5e9   interactive sky-cyan (links, focus)

-----------------------------------------------------------------------------
2. USAGE
-----------------------------------------------------------------------------
  - Keep clear space around the mark equal to the width of the tassel curl.
  - Don't recolor the mark beyond the two brand neutrals (navy / light) or a
    single accent color; use the binary marks for 1-color contexts.
  - Don't add gradients, drop shadows, or 3-D effects.
  - Minimum legible size: ~16px (favicon) for the mark, ~120px wide for the
    wordmark.

4. TRADEMARK: "AI Classroom"  (NOT legal advice - consult a TM attorney)
-----------------------------------------------------------------------------
From web search of USPTO-style records (not an authoritative clearance search):
  - No exact LIVE U.S. federal registration was found for the wordmark
    "AI CLASSROOM" in education/software classes.
  - Related/nearby marks show how crowded + hard-to-register this space is:
      * "LEARN WITH AI" (Serial 99018704, TIMEBACK LLC) - FINAL REFUSAL.
      * "INTERNATIONAL INSTITUTE OF AI EDUCATION" (99197318) - ABANDONED.
      * "AIBEP" (99404809) - pending, non-final office action.
  - Multiple businesses already trade under "Classroom AI" / "AI Classroom"-like
    names (apps + sites listed above) -> common-law rights + confusion risk.

Assessment:
  - "AI Classroom" is highly DESCRIPTIVE for an AI-taught classroom product.
    Descriptive marks are weak and typically refused on the Principal Register
    absent acquired distinctiveness (cf. the LEARN WITH AI refusal). Expect an
    uphill USPTO path and limited enforceability.
Recommendations:
  - Engage a trademark attorney for a full clearance search (USPTO TESS + state +
    common-law + WIPO/Madrid) before relying on the name.
  - Consider a distinctive/coined brand (logo + a unique wordmark) that is easier
    to register and defend; keep "AI Classroom" as a descriptive tagline. The
    new minimalist mark (this file) is significantly more registrable than the
    previous illustrated mark and should be filed as a DESIGN MARK.
