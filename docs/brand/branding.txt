-----------------------------------------------------------------------------
1. LOGO
-----------------------------------------------------------------------------
Concept: "fun but surreal." A friendly robot-owl teacher holds a green
chalkboard that melts and morphs into a swirling galaxy of chalk equations,
planets and a graduation cap, floating on a small surreal island. Dali-esque
surrealism in a clean, modern flat-vector emblem style.

Assets (docs/brand/):
  aiclassroom_logo.webp / .png        full-color emblem (primary)
  aiclassroom_logo_binary.png         1-bit B/W, dithered - best for grayscale
                                       print (newspaper/halftone look), keeps detail
  aiclassroom_logo_binary_threshold.png  1-bit B/W, hard threshold - bold 1-color
                                       "stamp/screen-print" mark (t-shirts, stickers,
                                       embossing, fax, single-ink printing)
Web (apps/web/public/):
  logo.webp        512px color (hero/marketing)
  logo-mark.webp   small mark used in the top nav
  favicon.ico      multi-size browser icon

Printing the binary version:
  - Both binary files are true 1-bit PNGs (mode "1"): black ink on white, no
    grayscale, so they print crisply on any printer and scale to any size.
  - Use *_threshold.png for single-color jobs (stamps, screen print, laser
    engraving); use *_binary.png when you want the shaded/detailed look.

2. DOMAIN: aiclassroom.com  (NOT verified live - do a registrar check)
-----------------------------------------------------------------------------
The sandbox has restricted network egress, so a live WHOIS/RDAP/DNS lookup could
not be performed here. From web search:
  - "aiclassroom.com" is a short, generic, high-demand .com - it is very likely
    ALREADY REGISTERED (often parked / privacy-masked or held for aftermarket
    sale). Treat it as probably unavailable or premium-priced.
  - Confusingly similar names are already in active use by OTHER parties:
      aiclassroom.world (AI Classroom World), nextsteps-aiclassroom.com
      (ClassInclined "AI Classroom Station"), classroom-ai.com / "Classroom AI"
      (Google Play app), classroomai-sifocom.net (3I Academy "Classroom-AI").
  Action items:
  - Verify availability/owner via ICANN RDAP (https://lookup.icann.org) or any
    registrar (GoDaddy/Namecheap/Cloudflare). If taken, consider a broker or an
    alternative TLD/name (see naming below).
  - Also secure matching social handles + defensive TLDs (.ai, .app, .edu-style).

-----------------------------------------------------------------------------
3. TRADEMARK: "AI Classroom"  (NOT legal advice - consult a TM attorney)
-----------------------------------------------------------------------------
From web search of USPTO-style records (not an authoritative clearance search):
  - No exact LIVE U.S. federal registration was found for the wordmark
    "AI CLASSROOM" in education/software classes.
  - Related/nearby marks show how crowded + hard-to-register this space is:
      * "LEARN WITH AI" (Serial 99018704, TIMEBACK LLC) - FINAL REFUSAL.
      * "INTERNATIONAL INSTITUTE OF AI EDUCATION" (99197318) - ABANDONED.
      * "AIBEP" (99404809) - pending, non-final office action.
  - Multiple businesses already trade under "Classroom AI" / "AI Classroom"-like
    names (apps + sites listed above) -> common-law rights + confusion risk.

Assessment:
  - "AI Classroom" is highly DESCRIPTIVE for an AI-taught classroom product.
    Descriptive marks are weak and typically refused on the Principal Register
    absent acquired distinctiveness (cf. the LEARN WITH AI refusal). Expect an
    uphill USPTO path and limited enforceability.
Recommendations:
  - Engage a trademark attorney for a full clearance search (USPTO TESS + state +
    common-law + WIPO/Madrid) before relying on the name.
  - Consider a distinctive/coined brand (logo + a unique wordmark) that is easier
    to register and defend; keep "AI Classroom" as a descriptive tagline.
  - Candidate distinctive names to clear: e.g. an invented coinage paired with the
    owl mark; file the LOGO (design mark) which is more registrable than the
    descriptive words alone.

  - Keep clear space around the emblem (>= the height of the graduation cap).
  - Don't recolor the color emblem; for 1-color contexts use the binary marks.
  - Minimum legible size for the full emblem ~ 28-32px tall (nav uses 30px).
