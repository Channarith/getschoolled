CHANGELOG - Agentic Online Education Platform
=============================================
Format: newest first. No markdown per project convention.

- Mobile multi-language UI (i18n): the entire AI Classroom mobile app is now
  translated into 13 languages out of the box (English, Spanish, French,
  German, Italian, Portuguese, Russian, Arabic, Hindi, Chinese, Japanese,
  Korean, Vietnamese), with the other 13 platform-supported languages wired
  into the picker as graceful fallback-to-English. New apps/mobile/src/i18n/
  package: `languages.ts` (locale registry, RTL flags), `strings.ts` (full
  hand-translated string dictionary), `index.tsx` (LocaleProvider, useT()
  hook, AsyncStorage-backed persistence, expo-localization device-locale
  detection, English fallback for missing keys). Every screen + BottomTabs
  routes through `t(...)`. Settings gains a LANGUAGE section with a
  flag-chip language picker (single-select, instant re-render). RTL locales
  apply `direction: rtl` to the root view. Backend localized too:
  aoep_shared.notifications builds rendered titles/bodies in 13 languages
  via the new locale param; curriculum service: GET /notifications/feed
  takes `locale=`, GET /notifications/locales lists supported codes;
  unknown locales fall back to English cleanly. expo-localization added.
  Tests: 4 new shared (test_notifications.py) + 3 new API
  (test_notifications_api.py); 94 + 291 backend tests pass. Mobile
  typecheck clean, Metro bundles iOS (1.81 MB / 698 modules) + Android
  (1.82 MB / 701 modules) + web (≈580 kB).

- Mobile Netflix-style features + push/local notifications: apps/mobile is now
  a real multi-screen app (5-tab bottom nav: Home, Drive, My List, Alerts,
  Settings). New screens: HomeScreen with rails (Continue Listening, My List,
  New This Week, Picked for You, Trending, Categories), MyListScreen,
  NotificationsScreen (server-rendered inbox + read-state), SettingsScreen
  (notification toggles, daily-reminder hour picker, permission request,

  permission request + Android channel, repeating daily class reminder,
  staggered "new class" / "picked for you" alerts, immediate "class complete"
  alert; foreground notifications surface as an in-app top banner that
  deep-links into Drive Mode. Continue Listening + My List + streak persist
  via @react-native-async-storage/async-storage. New Banner / Rail /
  BottomTabs components. Backend: aoep_shared/notifications.py builds a
  personalized notification feed (new_class / continue / recommended /
  streak / daily reminder) deterministically from the audio catalog and
  optional student state; curriculum service exposes GET /notifications/feed
  with student_id / interests / in_progress / completed / streak_days
  query params. Tests: 5 unit (aoep_shared.notifications) + 5 API
  (curriculum). End-to-end: mobile typecheck clean, Metro bundles iOS
  (1.76 MB / 693 modules) + Android (1.78 MB / 696 modules), live
  /notifications/feed returns rich personalized items.
